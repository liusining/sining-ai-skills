# Processing the work-on-todo Output

After running `work-on-todo`, the script returns a structured prompt. Follow these instructions:

1. Read the **Unresolved Issues** — these are the requirements to work on
2. Reference **Previous Results** and **Previous Findings** before proceeding
3. Work on the unresolved issues
4. While working:
   - Add new findings or issues to the "Investigation and Problems" section of the note
   - Add results aligned with targets to the "Target" section
5. When finished or blocked, call `commit` with the requirements you completed
